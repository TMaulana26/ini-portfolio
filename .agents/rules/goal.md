# Tujuan Proyek: Personal Portfolio Website

## Visi Aplikasi
Sebuah website portofolio pribadi yang simple, efisien, dan berkarakter untuk menampilkan keahlian pengembang web penuh (Full-Stack Web Developer). Website ini menggunakan pendekatan desain retro/neobrutalism minimalis yang bersih.

## Prinsip Aplikasi
- **Simple & Terfokus:** Desain tidak boleh terlalu ramai. Fokus utama adalah keterbacaan data pengalaman kerja dan proyek web.
- **Dinamis (API Ready):** Struktur halaman dirancang untuk menerima data proyek secara dinamis dari API backend. Pada Fase 1, frontend akan menggunakan data tiruan (mock data) yang strukturnya sama persis dengan response API asli.
- **Aksesibilitas Tinggi:** Antarmuka harus responsif, ringan saat dimuat, dan mendukung multi-bahasa (i18n) sejak awal.
